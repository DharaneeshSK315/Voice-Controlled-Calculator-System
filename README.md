# 🎙️ VoxCalc: Voice-Controlled Calculator System

[![Deploy Static Site to Pages](https://github.com/DharaneeshSK315/Voice-Controlled-Calculator-System/actions/workflows/static.yml/badge.svg)](https://github.com/DharaneeshSK315/Voice-Controlled-Calculator-System/actions/workflows/static.yml)

**VoxCalc** is a premium, voice-activated calculator application designed for a seamless, hands-free calculation experience. Built with a modern glassmorphism aesthetic and powered by high-accuracy speech recognition, VoxCalc is perfect for quick, reliable calculations through natural language.

---

## ✨ Features

-   **Voice-Driven Commands**: Speak naturally (e.g., "ten plus twenty", "five multiplied by six").
-   **High Accuracy Speech-to-Text**: Powered by the Web Speech API for real-time transcription.
-   **Audio Feedback (TTS)**: The system automatically reads out results, ensuring you don't miss any answers.
-   **Premium Modern Design**: Sleek indigo and purple theme with glassmorphism effects and neon accents.
-   **Hybrid Keypad Input**: Smooth, responsive touch/click-based keypad for traditional use.
-   **SEO Optimized & Mobile Responsive**: Optimized for any device and size.

## 🚀 How to Use

1.  **Open the Live Site**: [View Live Demo](https://dharaneeshsk315.github.io/Voice-Controlled-Calculator-System/)
2.  **Allow Mic Permissions**: Ensure you grant the browser access to your microphone.
3.  **Click the Mic Icon**: Tap the pulsing microphone button to start listening.
4.  **Speak your Calculation**: Try saying "twelve divided by three" or "one hundred minus fifty".
5.  **Get Results**: The expression will be visualized instantly, and the result will be calculated and spoken aloud.

## 🎙️ Supported Commands

| Operation | Examples |
| :--- | :--- |
| **Addition** | "plus", "add", "and" |
| **Subtraction** | "minus", "subtract", "take away" |
| **Multiplication** | "times", "into", "multiplied by" |
| **Division** | "divided by", "over", "divide" |
| **Utility** | "clear", "reset", "point" (for decimals) |

## 🛠️ Technology Stack

-   **Structure**: Semantic HTML5
-   **Styling**: Vanilla CSS3 (Advanced Flexbox, Grid, Glassmorphism, Animations)
-   **Logic**: Pure JavaScript (ES6+)
-   **APIs**: Web Speech API (SpeechRecognition & SpeechSynthesis)
-   **Icons**: Font Awesome 6
